Description du projet

Ce projet a été réalisé dans le cadre de mon stage chez Digi Activity afin d’apprendre les bases du Python. Il s’agit d’une reproduction du jeu de société Mastermind, dans lequel le joueur doit deviner un code secret généré aléatoirement.

J’ai implémenté la génération du code aléatoire, l’interaction avec le joueur via le terminal, la comparaison entre la proposition et le code secret, ainsi que la gestion du nombre d’essais restants.
