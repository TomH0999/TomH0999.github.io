Description du projet

Dans ce projet réalisé dans le cadre de mon stage Digi Activity pour apprendre les bases du Python, j'ai reproduit le jeu de société Masterming où l'on doit deviner un code couleur secret.

Ici, j'ai codé la génération d'un code aléatoire, l'intéraction avec le joueur pour qu'il propose une combinaison, la comparaison des deux codes et la gestion du nombres d'essais restants.
